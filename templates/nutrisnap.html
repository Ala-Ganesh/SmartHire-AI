NutriSnap-X

AI-based food recognition and nutrition estimation system

📌 Project Overview

NutriSnap-X is an AI-powered application that detects food items from images and estimates their nutritional values using machine learning.

🚨 Problem Statement

People often do not know the nutritional values of their daily food intake, which leads to unhealthy eating habits.

💡 Solution

NutriSnap solves this by using AI to recognize food items from images and provide nutritional insights instantly.

🛠 Technologies Used

  • Python
  • Flask
  • TensorFlow / MobileNetV2
  • OpenCV
  • NumPy
  • HTML, CSS, JavaScript

⚙ Tools & Libraries

  • VS Code
  • GitHub
  • Render (Deployment)

🏗 System Architecture

User uploads image → Image processed → AI model predicts food → Nutrition calculated → Result displayed

🚀 Deployment

Deployed using Render cloud platform as a web application.
