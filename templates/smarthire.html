SmartHire AI

AI-Powered Resume Portfolio for Recruiters

📌 Project Overview

SmartHire AI is an interactive web-based portfolio that allows recruiters to explore a candidate’s profile using an AI chatbot, project dashboards, and live demos. It replaces traditional resumes with an engaging and intelligent web application.

🚨 Problem Statement

Traditional resumes are static and do not effectively showcase a candidate’s real skills, projects, and capabilities.

💡 Solution

SmartHire AI provides a dynamic platform where recruiters can interact with an AI chatbot, explore projects, view GitHub activity, and understand the candidate deeply.

🛠 Technologies Used

  • Python
  • Flask
  • HTML, CSS, JavaScript
  • Chart.js

⚙ Tools & Platforms

  • VS Code
  • GitHub
  • Render (Deployment)

🏗 System Architecture

User → Web Interface → Flask Backend → AI Brain (NLP) → Response Display

🚀 Key Features

  • AI Chatbot for recruiter interaction
  • GitHub Dashboard Integration
  • Skills Visualization Charts
  • Project Showcase
  • Resume Download

🚀 Deployment

Deployed as a web application using Render cloud platform.
