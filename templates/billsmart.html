BillSmart

Smart Bill Reminder System

📌 Project Overview

BillSmart is a web-based application designed to help users track and manage their bill payments efficiently by providing timely reminders.

🚨 Problem Statement

Many people forget to pay bills on time, leading to penalties and inconvenience.

💡 Solution

BillSmart allows users to add bills, set due dates, and receive reminders to ensure timely payments.

🛠 Technologies Used

  • Python
  • Flask
  • HTML, CSS, JavaScript
  • SQLite Database

⚙ Tools & Platforms

  • VS Code
  • GitHub

🏗 System Architecture

User → Web Interface → Flask Backend → Database (SQLite) → Reminder System

🚀 Key Features

  • Add and manage bills
  • Set due date reminders
  • User-friendly interface
  • Data storage using database

🚀 Deployment

Can be deployed as a web application using Flask and cloud platforms.
