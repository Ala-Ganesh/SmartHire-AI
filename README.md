## 🌐 Live Demo

👉 https://alaganesh-portfolio.onrender.com

## 📌 About the Project

SmartHire AI is an interactive portfolio web application designed to showcase my skills, projects, and experience in a dynamic way.

Instead of a traditional resume, this platform allows recruiters to:

* Explore my projects with live demos
* View my technical skills
* Interact with an AI-powered chatbot
* Contact me directly

---

## ✨ Key Features

* 🤖 AI Chatbot Assistant (Ask about my profile, skills, and projects)
* 📊 Projects Showcase with Live Demo Links
* 🧠 Data Science & AI Focused Portfolio
* 🎨 Modern UI (Glassmorphism + ChatGPT-style Chat UI)
* 📱 Fully Responsive Design
* 🌐 Deployed Web Application

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Backend:** Python (Flask)
* **Database:** SQLite (optional)
* **AI Integration:** Custom Chat Logic / API Ready
* **Deployment:** Render

## 🚀 How to Run Locally

```bash
git clone https://github.com/Ala-Ganesh/SmartHire-AI.git
cd SmartHire-AI
pip install -r requirements.txt
python app.py
```

Open in browser:
http://127.0.0.1:5000
